**MANUAL DE USUARIO**

*Instalación del Servidor*1.Abrir el proyecto ServidorRecomendador en Netbeans.2.Acceder a la clase Recomendador.java e indicar en las líneas 125 y 153 el directorio donde se quiere guardar las bases de datos.3.Pulsar en la opción Clean and Build para compilar el proyecto.4.Para lanzar el servidor pulsar en la opción Deploy.5.En el menú de la izquierda abrir la carpeta Web Service del proyecto y sobre el elemento Recomendador pinchar en la opción Test Web Service. Se abrirá en el navegador una pestaña nueva con las operaciones implementadas en el servidor.6.Pinchar en la opción actualizarBaseDatos() para cargar las bases de datos en el servidor.7.Ya esta funcionando el servidor. Cuando se quiera apagar el servidor pinchar en opción saveData() para guardar el estado del grafo para la próxima vez.8.Para las siguientes veces que se quiera lanzar el servidor solo será necesario realizar el paso 4, y 5. Y una vez en el navegador pinchar en la opción loadData() para cargar el grafo en la memoria del servidor. *Uso del Editor de Grafos*1.Abrir el proyecto EditorGrafos en NetBeans y ejecutar el proyecto. También se puede ejecutar el programa de forma manual desde el terminal compilando los archivos con la orden javac *.java y luego ejecutar el editor con la orden java EditorGrafos. Se abrirá una ventana con el editor de grafos.2.Editar el grafo con las distintas opciones que se incluyen en el panel.   2.1.Opción Node: Pinchar en el punto del mapa donde se quiera añadir un nodo.   2.2.Opción Link: Pinchar el nodo que se quiera como origen del enlace, dicho no se pondrá en verde. Posteriormente pinchar en el nodo que se quiera como nodo destino.   2.3.Opción Move: Pinchar el nodo que se quiera cambiar de posición en el mapa, dicho nodo se pondrá en verde. Posteriormente pinchar en la nueva localización del mapa.   2.4.Opción Delete Node: Pinchar sobre el nodo que se quiere eliminar.   2.5.Opción Delete Link: Pinchar el nodo origen del enlace que se quiere borrar, dicho nodo se pondrá en verde. Posteriormente pinchar el nodo destino. 3.Ejecutar la clase getBaseDatos.java del paquete basedatos desde Netbeans o de forma manual desde el terminal ejecutando la orden javac getBaseDatos.java. Esto hará que se creen las bases de datos adaptadas para el Servidor y la Simulación.4.Acceder al Servidor desde el navegador a través del Test Web Service y ejecutar el método actualizarbaseDatos(). Esto hará que se actualicen las bases de datos en el Servidor y en el Simulador.*Ejecución del Simulador*1.Arrancar el ServidorRecomendador.2.Abrir el proyecto Simulacion en NetBeans y ejecutar el proyecto. También se puede ejecutar el programa de forma manual desde el terminal compilando los archivos con la orden javac *.java y luego ejecutar la simulación con la orden java Simulacion. Se abrirá una ventana con la simulación.3.Iniciar el programa pulsando el botón Iniciar. 4.El simulador permite seleccionar diferentes opciones.   4.1.Pinchar la el botón Pausar/Reanudar para poner en pausa o seguir con la simulación.   4.2.Se puede elegir varias opciones de tipo de trafico (Bajo, Medio y Alto).   4.3.Pinchar en la opción Mostrar Grafo para representar  el grafo sobre el mapa.   4.4.Para crear un vehículo cliente introducir el nodo origen y el nodo destino en las casillas correspondientes y pulsar sobre el botón Crear Vehiculo. Esto hará que se creen dos vehículos que seguirán la ruta descrita. El primer vehículo, de color verde, será el que represente el que usa el servicio de recomendación de ruta según trafico y el segundo vehículo, de color azul, será el que represente el que se basa solamente en la distancia.   4.5.Para indicar una incidencia y simular el corte de un tramo de carretera, introducir el nodo origen y el nodo destino en las casillas correspondientes y pulsar sobre el botón Cortar calle. Para volver a poner la calle en estado de normalidad pulsar sobre el botón restablecer.5.Para reiniciar la simulación pulsar sobre el botón Reiniciar.
